JPHP - это новая реализация PHP, использующая Java VM. Она поддерживает многие возможности языка PHP (5.6+).

Как это работает? JPHP это компилятор как javac, он компилируте php исходники в JVM байткод и после
может быть выполнен результат на Java VM.

Достижения

JPHP is not a replacement for the Zend PHP engine or Facebook HHVM. We don’t plan to implement the zend runtime libraries (e.g. Curl, PRCE, etc.) for JPHP.

Разработка DevelWare началась в 2024 году. Пять причин использовать JPHP:

  1. Умение использовать java-библиотеки в PHP
  2. Повышение производительности с помощью JIT и JVM
  3. Замена уродливой библиотеки времени выполнения Zend PHP на более совершенную библиотеку времени выполнения.
  4. Использование языка PHP не только в веб-сайтах
  5. Также: юникод для строк и потоков
